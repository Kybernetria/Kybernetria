There is no such thing as a neutral algorithm. The mathematical models deployed by the carceral state and corporate monopolies are ideological constructs masked as objective mathematics. Predictive policing does not forecast crime; it forecasts policing. It creates a feedback loop that validates its own historical biases, directing violent state resources into already over-policed neighborhoods.

When tech companies claim their AI systems are simply "optimizing" resource distribution, we must ask: optimizing for whom?

## The Illusion of Objectivity

<!-- PLACEHOLDER: deeper analysis on machine learning -->
Machine learning models are trained on historical data. If that history is defined by structural racism and class warfare, the model will faithfully reproduce those conditions. To the system, this is a successful prediction. To the community, it is a self-fulfilling prophecy of state violence.

### Re-appropriating the Network

But we are not powerless. The same networked infrastructure used to surveil can be repurposed to coordinate resistance. The concept of *détournement*—the situationist practice of hijacking capitalist media for radical purposes—must be extended to software architecture.

1. Build encrypted, federated communication networks
2. Dismantle the myth of the "unbiased machine" in public discourse
3. Develop counter-algorithms that map state power rather than marginalized communities

We do not need smarter algorithms to govern us; we need the power to govern our algorithms.
